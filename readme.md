# Free Mind map

Бесплатная рисовлка интеллект-карт в браузере


## STAGES:

### mindMap 0.1:
Frontend на JS, Kotlin Backend, Spring




ТЗ:
При входе на страницу пользователь видит canvas, в котором находится один созданный узел. Главных узлов может быть много, несколько
mindmap-ов могут находиться в одном canvas и могут быть не связаны друг с другом.

1) Узлы представляют собой прямоугольники со скругленными краями, при наведении на узел должна появляться стрелка для создания дочерней, а также маленькая палитра цветов
2) Должна быть возможность удалять узлы по нажатию Backspace, если пользователь не редактирует текст узла. Если пользователь выбрал узел и начинает вводить текст, то этот текст должен вводиться внутрь узла. 
3) Должна быть возможность удалять стрелки между узлами
4) Должна быть возможность перетаскивать узлы
5) Должна быть возможность вводить много текста для узлов, а также уменьшать узлы с большим текстом при наведении на узел, при уменьшении должен отображаться только текст первой строки
6) в правом нижнжем углу canvas должны быть кнопки для уменьшения и увеличения масштаба, а также кнопка для возврата к стандартному значению масштаба, кнопка для центрирования камеры по выделенному узлу, либо по всей карте если выделенных узлов нет, также кнопка для автоматического выравнивания всех узлов, кнопки undo и redo
7) должна быть возможность выделять и копировать и вставлять множество узлов
8) При нажатии кнопок ctrl+e (export) должно выходить модальное окно для сохранения карты в формате png и json
9) при нажатии кнопок ctrl+i (import) должно выходить окно куда можно ввести json и по нему отрисуется карта
10) при нажатии кнопок ctrl+h (help) должно выходить окно с подсказками
11) должна быть возможность перемещения внутри canvas стрелками
12) Дизайн должен быть минималистичным, линии тонкими и аккуратными
13) Код приложения должен быть максимально понятным и блочным, чтобы можно было легко его редактировать и дополнять, даже если человек плох в js
